Case 01 · AI Agent

AI Agent 招聘流程自动化

背景：B 端招聘交付中存在大量重复性工作，包括简历筛选、候选人信息整理和交付流程跟进，人工处理耗时较高。

问题：岗位筛选标准分散，候选人匹配过程依赖人工经验，团队难以快速复用统一处理流程。

方案：将招聘交付拆解为岗位要求识别、候选人信息提取、匹配判断和结果输出四个环节，基于 Claude 搭建 AI Agent 工作流，并完成本地化部署和可视化页面。

价值：单份处理时间由 15min 降至 5min，流程效率提升 3 倍，支持团队共享使用。

Case 02 · Ad Operations

广告履约监控与补量策略库

背景：效果广告交付过程中，履约进度受客户规模、广告位特性和流量波动影响，延期风险需要提前识别。

问题：原有监控更多依赖结果复盘，异常出现后再处理，缺少提前预警和标准化补量策略。

方案：搭建流量履约压力监控表，按客户规模和广告位特性分类沉淀补量策略，结合 SQL 和 Excel 完成点击数据采集、清洗和异常 case 归因。

价值：策略库覆盖 3 类客户场景，头部客户履约率 90%+，延期履约用户数下降 60%。
